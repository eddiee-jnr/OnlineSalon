# Online Salon Booking

This repository contains the front-end of an online salon booking system. The project includes HTML, CSS, JavaScript, and assets to support user interactions such as browsing products and services, booking appointments, and managing a shopping cart.

## Project Structure

- HTML files representing various pages: `index.html`, `booking.html`, `products.html`, etc.
- CSS stylesheets located in `styles/` for layout and design.
- JavaScript files located in `js/` implementing page behavior and data operations.
- Image and media assets under `assets/images` used across the site.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/eddiee-jnr/OnlineSalon.git
   cd OnlineSalon
   ```
2. Open any HTML file in a browser to view the pages. Tools like Live Server (VS Code) can serve the project for rapid development.
3. Modify HTML, CSS, or JS files as needed, then commit and push changes.

## Features

- Home page with promotional banners and navigation.
- Booking page for scheduling appointments.
- Product pages showcasing salon products.
- Cart and checkout workflow.
